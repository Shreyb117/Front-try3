About

This course rather than focussing just on the theory and the traditional way of teaching, focuses more on learning through examples. This problem solving approach of studying leads to a better understanding of real world applications. This module based curriculum is an intriguing approach in Educational research. We have developed a course that is not centered exclusively on theory or the code snippets and is rather somewhere between both these two extremes. This is pretty much the rationale behind this project. For the purpose of this project, we are using the Python language to write the code as python has some libraries like NumPy and pandas which we can harness to achieve various functionalities essential for a course like this one.

About the Dependency Graph


To structure the project, we are in the process of making a dependency graph which allows the user to effectively study at their own pace and prerequisite specifications. This graph is made with the use of a python library known as graphviz.

  • Adependencygraphisanetworkthatpresentsthelearningforacourseby presenting a concept as a node.
  • Every node is connected to several other nodes depending on the prerequisites required to understand the concept.
  • Similarly,this node is then connected to several other nodes to form a kind of roadmap for the course.
  • Furthermore, each node is a hyperlink which leads to several examples required for understanding the concept.
  • This idea of a dependency network is basically based on the premise that knowledge is a network of connected nodes.
